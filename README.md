# Verificador de Idade

Projeto simples em **HTML, CSS e JavaScript** que calcula a idade pelo **ano de nascimento**, identifica o **sexo** e exibe um resultado com imagem (criança, adolescente, adulto(a), idoso(a)).

## Como usar
1. Baixe/clone o repositório
2. Abra o arquivo `index.html` no navegador

## Estrutura
- `index.html` → página principal  
- `estilo.css` → estilos  
- `script.js` → lógica do verificador  
- `*.png` → imagens usadas no resultado

## Observação
Deixe as imagens na mesma pasta do `index.html` com os nomes:
`crianca.png`, `adolescente.png`, `adolescentem.png`, `adulta.png`, `adulto.png`, `idosa.png`, `idoso.png`.
